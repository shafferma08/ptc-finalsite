Build notes

Brand-compliant: Colors, body font (Roboto), and headline font (Roboto Slab) match the Brand Guide Sheet's free Google Font alternatives. Swap to Elizeth + Proxima Nova where Adobe fonts are licensed.

Script font flag: The brochure's hand-lettered accent ("and it all starts now.") isn't specified in the brand guide. Caveat Brush is the closest free Google approximation; worth confirming with your graphic designer or requesting the actual typeface.

Network pattern: Recreated as inline SVG so it scales cleanly and ships with no image dependency. Designer's original vector would be a drop-in replacement.

Photography: The four program cards use images extracted from the brochure PDF. For production, request source resolution from the designer.

Finalsite port: Each component is a self-contained block with no layout dependencies beyond the shared tokens, so they should drop into Composer elements cleanly. Band, hero, pill, program card, and step module can each be a reusable Composer component.
