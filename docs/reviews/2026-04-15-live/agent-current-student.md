## Conclusion

**For a busy, impatient current student like Maria:**

The PTC website serves recruitment and information seekers well (bright colors, program cards, news). But it fails to prioritize existing students' most urgent needs: getting to their classes, accessing Canvas, and finding campus essentials.

**Key Insight:** Maria doesn't use this site to GET to PTC—she already IS at PTC. She uses it as a quick reference tool or entry point to Canvas/Focus. The current hierarchy (Recruitment > Programs > News > Student Links) doesn't match her mental model.

**Recommended Priority:**
1. Make Canvas 1-click from homepage (button, not dropdown menu)
2. Add a "Current Students" homepage section with Canvas, Focus, resources
3. Create a "My Schedule" page or Campus Calendar that shows student cohort schedules
4. Publish campus hours prominently

These changes would transform the site from "College Information Portal" into "Student Hub"—which is what Maria actually needs.
