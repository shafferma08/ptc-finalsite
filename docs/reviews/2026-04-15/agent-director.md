# Director/Admin Review - 2026-04-15

## Executive Summary
Overall, the site demonstrates solid foundational structure and clear institutional identity. However, several critical gaps exist that require immediate attention before this can confidently represent PTC to accreditation bodies, board members, and the public. Primary concerns center on incomplete implementation, missing compliance elements, and inconsistencies between pages.

---

## Institutional Accuracy

### Strengths
- Clear institutional identity: PTC brand (green #008142) consistently applied across all pages
- Mission statement properly displayed on About page with appropriate framing
- Accurate program clustering across both campuses with campus-specific differentiation
- Contact information current and correct (Clearwater 727.538.7167, St. Pete 727.893.2500)
- Academic calendar, Canvas, Focus portal properly referenced
- Founded date (1961) and 60+ years messaging accurate

### Critical Issues
1. **Leadership Representation Incomplete**
   - About page shows placeholder "Campus Director" icons with no actual names, titles, or contact info
   - Leadership grid (3 slots: Clearwater Director, St. Pete Director, District Admin) contains only icons and generic titles
   - No "View Full Staff Directory" link is functional
   - **Risk**: Accreditation site visit will expect full leadership visibility and chain of command

2. **Program Accuracy Concerns**
   - Programs.html claims 40+ programs but lists approximately 35-38 distinct offerings
   - Some programs listed with inconsistent naming (e.g., "Practical Nursing" vs variations across pages)
   - Missing explicit OCP (Occupational Completion Point) designation for certificate vs. diploma programs
   - **Risk**: Misleading prospective students and auditors on actual program count

3. **Missing Program Details**
   - No program descriptions, duration, tuition, or employment outcomes visible on any page
   - Program cards hyperlink to "#" rather than actual program detail pages
   - No industry certifications or credentialing bodies listed for individual programs
   - **Institutional Risk**: COE accreditation requires documented program learning outcomes and employment data

## Top 3 Issues (Ranked by Institutional Risk)

### PRIORITY 1: Leadership Representation Gap (CRITICAL)
**Issue**: The About page shows only generic placeholder icons for "Campus Director" (both campuses) and "District Administration" with no actual names, titles, or contact information.

**Why This Matters Institutionally**:
- COE accreditation evaluators will expect to see institutional leadership clearly identified
- PCSB board members visiting the site expect campus leadership visibility
- Prospective students and families need to know who runs the institution
- If this site is shown during accreditation visits, the absence of leadership messaging raises serious questions about transparency and governance

**Impact**: High institutional risk
**Timeline to Fix**: Immediate (before any board presentations or accreditation visits)
**Action**: Add actual names, titles, photos, and contact information for:
- Clearwater Campus Director
- St. Petersburg Campus Director  
- District-level PTC administrator (if applicable)

---

### PRIORITY 2: Broken Application & Financial Aid Pathway (HIGH)
**Issue**: All "Apply Now," "Start Your Application," and enrollment-related CTAs link to "#" rather than the actual application portal (https://apply.myptc.edu/). Financial aid information is missing entirely (no tuition page, no FAFSA link on main site, no cost of attendance).

**Why This Matters Institutionally**:
- The site directs interested prospects to apply, but the application button doesn't work
- Financial aid is a critical enrollment driver; missing info will reduce applications
- PCSB leadership and board members will view this as an incomplete tool for enrollment conversion
- Enrollment numbers drive funding; broken CTAs directly impact institutional metrics

**Impact**: Very High institutional risk (directly affects enrollment)
**Timeline to Fix**: Before next recruitment season (urgent)
**Action**: 
1. Ensure all Apply/Application CTAs link to active portal
2. Create dedicated Financial Aid page with FAFSA info, School Code (005605), aid types, Net Price Calculator
3. Add tuition rates and program-specific cost information
4. Create clear enrollment pathway visualization

---

### PRIORITY 3: Missing Compliance Documentation Links (HIGH)
**Issue**: Multiple footer links (Privacy Policy, Accessibility, Consumer Information, Non-Discrimination) are referenced but not functional. No links to accreditation reports, program outcomes data, or PCSB governance documents.

**Why This Matters Institutionally**:
- COE auditors expect to find accreditation documentation readily available
- Federal compliance (Consumer Information Act, FERPA, ADA/504) requires accessible statements
- PCSB governance oversight requires clear policy visibility
- If accreditation visitors cannot find required compliance info, it signals institutional disorganization

**Impact**: High institutional risk
**Timeline to Fix**: Before accreditation interactions (1-2 weeks)
**Action**:
1. Create and link Privacy Policy document
2. Create and link Accessibility statement (WCAG 2.1 AA compliance claim)
3. Create Consumer Information page (per U.S. Department of Education requirements)
4. Create Accreditation page with links to:
   - Most recent COE accreditation report (both campuses)
   - Cognia/district accreditation overview
   - Program-level learning outcomes and placement rates
5. Ensure Non-Discrimination policy document is downloadable

---

## Recommendations for Board Presentation

If this site is to be shown to PCSB leadership or accreditation visitors, the following should be addressed first:

1. **Before Board Meeting**: Add actual leadership names/titles to About page; ensure all CTAs are functional
2. **Before Accreditation Visit**: Complete all compliance links and create accreditation documentation page
3. **Medium Term**: Complete program detail pages; add outcomes/placement data for each program cluster
4. **Ongoing**: Implement analytics to track enrollment conversion; monitor time-to-apply and drop-off points

The site has strong visual design and good structural organization. With these institutional gaps filled, it will be ready for public-facing and accreditation purposes.
