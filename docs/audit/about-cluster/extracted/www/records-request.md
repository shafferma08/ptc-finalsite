(How to) Request Your Student Record(s) from PTC

Request for School Records

To request a copy of your class records, download the class records that must be completed.  After you have completed the Request for School Records and Release Authorization form mail it to the PTC campus where you attended.

Pinellas Technical College - Clearwater
Attn: Records Department
6100 154th Ave North
Clearwater, FL 33760

Clearwater: Request For School Records and Release Authorization Form
email to canfieldj@pcsb.org
